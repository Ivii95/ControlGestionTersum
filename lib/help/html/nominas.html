Nominas.

Ventana en la que se muestran las nominas. Puede añadirlas, modificarlas y borrarlas.

Para añadir solo rellena los campos y pulsa añadir.

una vez que haya añadido se le redigira a una pagina para asignarla a un trabajador.

Para modificar seleccione una nomina en la tabla y modifique el campo que desee.

Para borrar seleccione una nomina y pulse el boton borrar.

Pulse el boton limpiar para retroceder y que le salga el boton añadir.
