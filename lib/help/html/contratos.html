Contratos.

Ventana en la que se muestran los contratos. Puede añadirlos, modificarlo y borrarlo.

Para añadir solo rellena los campos y pulsa añadir.

una vez que haya añadido se le redigira a una pagina para asignarlo a un trabajador.

Para modificar seleccione un contrato en la tabla y modifique el campo que desee.

Para borrar seleccione uno y pulse el boton borrar.

Pulse el boton limpiar para retroceder y que le salga el boton añadir.
